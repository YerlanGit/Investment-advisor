## Рекомендация по гигиене (требует подтверждения — не выполнено автоматически)

Файлы-снимки 🗄 (`REPORT_SECTIONS_AUDIT.md`, `docs/BANK_RAG_COVE_DIAGNOSIS.md`) —
исторические диагностики на дату, их выводы уже реализованы. Они **перекрёстно
ссылаются из `AUDIT.md` и `docs/REPORT_PAGES_*`**, поэтому удаление/перемещение
оставит битые ссылки. Варианты, если хотите разгрузить корень:
1. Перенести в `docs/archive/` + поправить ссылки в `AUDIT.md`/`REPORT_PAGES_*`
   (обратимо, `git mv` сохраняет историю).
2. Оставить как есть — они лёгкие и несут audit-trail.

Дубли по смыслу отсутствуют: RAG-кластер (ingestion/troubleshooting/diagnosis)
и LLM-кластер (MCP/multilingual/system-prompt) покрывают РАЗНЫЕ грани, не
дублируют друг друга — объединять не рекомендуется.
