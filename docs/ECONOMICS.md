## 1. Стоимость ОДНОЙ генерации по каждому тиру

### 1.1 Модельная арифметика (LLM)

Оба тира делают **ровно один** вызов Anthropic API (структурированный tool-call `emit_report`); сценарный тир — **ноль** вызовов (полностью детерминированный).

Составные части промпта (замерено по коду):
- `SYSTEM_PROMPT.md`: 25 468 байт ≈ **10–12k токенов** (кириллица токенизируется плотнее; токенайзер Sonnet 5/Opus 4.8);
- user-prompt BASE: summary JSON ≤ 7 000 симв. + инструкции + RAG-контекст ≤ 2 000 симв. ≈ **4–6k токенов**;
- user-prompt DEEP: summary ≤ 9 000 симв. + RAG ≤ 6 000 симв. + инструкции ≈ **7–9k токенов**;
- output: BASE `max_tokens=5 000` (типично 3–4k), DEEP `max_tokens=7 000` (типично 4.5–6k).

Тарифы (Anthropic, 2026-07): **Sonnet 5 = $3 input / $15 output за 1M** (интро-цена $2/$10 до 2026-08-31); **Opus 4.8 = $5 / $25 за 1M**.

### 2.3 Немедленная оптимизация: prompt caching (не внедрён — L-12 аудита)
SYSTEM_PROMPT (~11k токенов) платится полностью в каждом вызове. С `cache_control: {type: "ephemeral"}` на system-блоке: запись ×1.25 один раз в 5 минут, чтение ×0.1.
- BASE: input $0.045 → ~$0.022 (−50%), итог ≈ $0.075/отчёт;
- DEEP: input $0.095 → ~$0.052, итог ≈ $0.19/отчёт.
При потоке отчётов чаще 1 раз / 5 мин экономия LLM-бюджета ≈ **35–45%**. Трудозатраты: ~20 строк в `ai_narrative`.

### 2.4 Точка безубыточности (тариф 2026-07-17: 1 токен = 2 500 ₸)
Фикс ≈ $65/мес → нужно ≈ **14 платных токенов/мес** (≈ 7 DEEP-отчётов), т.е. **~2 платящих пользователя** с пакетом «10 токенов за 25 000 KZT» в месяц (было ~8 при старом тарифе 500 ₸).

## 5. Следующие шаги масштабирования и их стоимость

### 5.1 База данных (обязательный шаг №1 — снимает потолок 1 инстанса)

|---|---:|---|
| Cloud SQL PostgreSQL db-f1-micro / db-g1-small | $9–35 | прямая замена SQLite (балансы, профили, vault), + Private IP |

| AlloyDB / Cloud SQL HA | $150+ | только при 10k+ MAU |

### 5.2 Telegram: long-polling → webhook
Стоимость $0 (тот же Cloud Run, наоборот дешевле: убирается `--no-cpu-throttling` и min-instances можно 0–1). Требует БД-миграции (5.1) и снятия max-instances=1. Экономия ≈ $30–40/мес на idle-CPU + горизонтальное масштабирование.

### 5.3 Данные: новостные и рыночные фиды
| Источник | Что даёт | $/мес |
|---|---|---:|
| NewsAPI.org | новости для нарратива/идей | $0 (dev) / $449 (business) |
| Finnhub | новости + фундаментал + альтернативные данные | $0 (rate-limited) / от $50 |
| Polygon.io | котировки/новости US real-time | $29–199 |

### 5.4 RAG / ChromaDB рост (больше банковских PDF)
- До ~50–100k чанков: текущая схема (Chroma на GCS + in-container) держит, стоимость = хранение GCS (~центы).
- Дальше: Vertex AI Vector Search (~$70+/мес минимум) или managed Qdrant/Pinecone ($50–100/мес starter). Триггер миграции: латентность retrieval > 1-2 с или образ > памяти инстанса.
- Рост бакетов Google (reports + chroma): $0.02/ГБ/мес — при 30k отчётов/мес ≈ 60 ГБ/год ≈ $1.2/мес; настроить lifecycle-policy авто-удаления отчётов старше 90 дней (TTL ссылок и так 48 ч).

### 5.5 Подключение MCP (Model Context Protocol)
Сценарий: вынести инструменты (Tradernet-цены, FRED, SEC, RAG-поиск) в MCP-серверы, чтобы LLM-слой мог вызывать их напрямую (agentic-режим вместо одного мегапромпта).
- **Инфраструктура:** 1 лёгкий Cloud Run сервис-MCP (0.5 vCPU, scale-to-zero) ≈ **$5–15/мес**.
- **LLM-эффект:** agentic-цикл с tool-calls увеличивает число вызовов на отчёт (2–5 вместо 1) → LLM-cost DEEP ≈ $0.4–0.8/отчёт, НО input сокращается (модель тянет только нужные данные) и качество идей растёт. Рекомендуется только для DEEP-тира или нового «PRO»-тира по $5+.
- **Разработка:** ~1-2 недели (обёртки уже структурированы: `finance/tool_plugins.py` — задел).

### 5.6 Telegram-каналы обновлений / push-аналитика (retention)
Ежедневный дайджест «что изменилось в портфеле» = 1 SCENARIO-подобный прогон без LLM (или Haiku 4.5 $1/$5 → ~$0.01/дайджест). При 1 000 подписчиков ежедневно: ~$300/мес на Haiku или ~$0 детерминированно. Драйвер удержания с почти нулевым COGS — рекомендую детерминированную версию.

### 5.7 Мониторинг и SRE (при 1k+ MAU)
Uptime-checks + alerting (Cloud Monitoring) $0–10; Sentry (errors) $0–26; бюджет-алерты Anthropic/GCP — бесплатно. Итого ≈ **$10–40/мес**.

| Этап | Разовые работы | Δ фикс-затрат / мес |

## 6. Ключевые риски экономики

1. **Цены Anthropic**: интро-тариф Sonnet 5 истекает 31.08.2026 (+50% к input/output BASE) — заложено в §1.
2. **Freedom Market Data**: подписка на исторические данные у брокера может стать платной для пользователей (уже сейчас требуется активация) — единственный внешний фид без бесплатной гарантии.
3. **Бесплатные 10 токенов** = ~$1 COGS-лимит на нового пользователя (≈ $0.5 при кэшировании) — приемлемо; НО с тарифом 2 500 ₸ розничная ценность welcome-бонуса выросла до 25 000 ₸ → стимул для фрод-ферм кратно выше: лимит по telegram_id уже есть, rate-limit на /start и антифрод-эвристики становятся обязательными до открытого запуска (см. PRODUCTION_READINESS P0).
4. **Курс KZT/USD**: выручка в тенге, затраты в долларах — при девальвации KZT маржа сжимается; после повышения до 2 500 ₸ запас прочности большой (маржа ~98% при курсе 520), но правило пересмотра цены при курсе > 560 сохраняем.
5. **Ценовой шок для беты (новое, 2026-07-17)**: рост цены ×5 (500 → 2 500 ₸) может уронить конверсию из бесплатных токенов — отслеживать долю пользователей, докупающих пакет после бонуса, и готовить промо-пакеты/подписку как амортизатор.
